# Prerequisites
- Node v18+ [Visit Google](https://nodejs.org/en/download)

# Installation
- To run this [Nuxt 3](https://nuxt.com/docs)
```
git clone https://github.com/Majd-Yahia/meteoweather-player.git
```
- Go into the directory
```
cd meteoweather-player
```
- Donwload node packages
```
npm install
```
- Start the websocket
```
npm run dev
```

# Important Note
Please make sure the [Websocket](https://github.com/Majd-Yahia/meteoweather-socket) is started before you run this project
